# GitHub Pages 预览

这是前端界面的 GitHub Pages 预览版本。

## 如何启用 GitHub Pages

1. 进入仓库的 Settings（设置）
2. 在左侧菜单中找到 "Pages"
3. 在 "Source" 下拉菜单中选择 "Deploy from a branch"
4. 在 "Branch" 下拉菜单中选择 "main" 或您的当前分支
5. 在文件夹下拉菜单中选择 "/docs"
6. 点击 "Save"

几分钟后，您的网站将在以下地址可用：
```
https://[您的用户名].github.io/cinema/
```

## 注意事项

- 这是纯前端预览版本，不包含后端功能
- WebSocket 聊天功能已替换为本地演示模式
- 完整功能需要部署包含 Node.js 后端和 SRS 流媒体服务器的完整版本
